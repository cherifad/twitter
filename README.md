## Deployment

Clone the repository

```bash
  git clone https://github.com/cherifad/twitter
```

Go to the 'docker' folder

```bash
  cd twitter/docker
```

Run the container

```bash
  docker compose up -d
```





Go to

```bash
  localhost:5173
```

ENJOY!
